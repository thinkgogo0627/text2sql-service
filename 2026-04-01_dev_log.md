**현상**

**원인**

- 실제 도커 postgres는 호스트 포트 **5433**으로 노출 (`docker-compose.yml`: `5433:5432`)

**해결**

### BUG-02: 프롬프트 포맷팅 KeyError
**현상**
```
Judge 실패, skip: '"sentiment_score"'
```

**원인**
```python
# run_eval.py:66
prompt = prompt_template.format(text=text_content[:3000])
```
- `news_content`에 `{"sentiment_score": ...}` 형태의 JSON 문자열 포함
- Python `str.format()`이 `{...}` 패턴을 포맷 플레이스홀더로 해석 → `KeyError`

**해결**
```python
prompt = prompt_template.replace("{text}", text_content[:3000])
```
